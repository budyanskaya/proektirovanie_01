# Задание 1. Используя «1С:Система проектирования прикладных решений» для выбранного предприятия, выполните классификацию бизнес-процессов.
### Сгруппируйте процессы по следующим пяти категориям:
* Базовые процессы
* Ограничивающие процессы
* Обеспечивающие процессы
* Процессы производства
* Процессы управления развитием
Для представленных процессов определите предшествующие процессы, где это релевантно.
### Для трёх выбранных процессов из разных групп выполните их подробное описание. В описании каждого процесса укажите:
* Краткое описание процесса
* Когда стартует процесс и чем завершается
* Шаги процесса — для каждого шага: дайте краткое описание, приведите примеры формулировки требований к системе (для некоторых шагов).
